## Install the Voice UI Kit

```bash tab="npm"
npm install @pipecat-ai/voice-ui-kit
```

```bash tab="pnpm"
pnpm add @pipecat-ai/voice-ui-kit
```

```bash tab="yarn"
yarn add @pipecat-ai/voice-ui-kit
```

```bash tab="bun"
bun add @pipecat-ai/voice-ui-kit
```

### Pipecat-specific dependencies

Depending on the nature of your project, you might need to install additional dependencies. If you are building using the Pipecat Client SDK, you will also likely need to install the Pipecat transport you are using:

```bash tab="Daily transport"
npm install @pipecat-ai/voice-ui-kit @pipecat-ai/daily-transport
```

```bash tab="SmallWebRTC transport"
npm install @pipecat-ai/voice-ui-kit @pipecat-ai/smallwebrtc-transport
```
